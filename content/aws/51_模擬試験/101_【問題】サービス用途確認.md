# 【問題】サービス用途確認（公式サンプル風・全10問）

## この問題集の目的

AWS Certified Solutions Architect - Associate の公式サンプルに近いシナリオ文で、主要サービスの役割を確認する。

## 対象レベル

SAA入口。単語暗記ではなく、要件からサービスを選ぶ練習にする。

## 問題数

10問

---

## 第1問（オブジェクトストレージ）
ある企業は、ユーザーがアップロードする画像とPDFを数年単位で保存する新しいWebアプリケーションを構築している。データはHTTP経由で取得でき、耐久性が高く、容量の増減に合わせたサーバー管理を避けたい。最も適切なサービスはどれか。

- A. Amazon S3
- B. Amazon EBS
- C. Amazon EFS
- D. Amazon FSx for Windows File Server

> [!success]- 答え：**A**
> S3は高耐久なオブジェクトストレージで、画像、PDF、バックアップ、ログなどの保存に適している。
> - A：正解。オブジェクト単位で保存し、容量管理の運用負荷が低い。
> - B：EC2にアタッチするブロックストレージで、HTTP経由の共有保存先としては第一候補ではない。
> - C：複数EC2から共有するLinuxファイルシステム用途。
> - D：Windowsファイル共有用途であり、汎用的なオブジェクト保存要件とは異なる。
> **関連ノート**：[[0201_S3]] / [[0202_S3とEBS・EFSの違い]]

## 第2問（マネージドRDB）
企業はオンプレミスで稼働しているPostgreSQLデータベースをAWSへ移行する。アプリケーションのSQLやトランザクション処理は大きく変えず、バックアップ、パッチ適用、フェイルオーバーの運用負荷を削減したい。最も適切なサービスはどれか。

- A. Amazon RDS
- B. Amazon DynamoDB
- C. Amazon Redshift
- D. Amazon Athena

> [!success]- 答え：**A**
> RDSはPostgreSQLなどのリレーショナルDBをマネージドで提供する。
> - A：正解。既存RDBの移行先として自然で、運用負荷を下げられる。
> - B：NoSQLデータベースであり、既存SQLアプリをそのまま動かす用途ではない。
> - C：分析用データウェアハウス。
> - D：S3上のデータをクエリするサーバーレス分析サービス。
> **関連ノート**：[[0301_RDS]]

## 第3問（イベント駆動処理）
アプリケーションは、S3にアップロードされた画像ごとにサムネイルを生成する必要がある。処理は短時間で終了し、常時稼働サーバーの管理を避けたい。最も適切なコンピューティングサービスはどれか。

- A. AWS Lambda
- B. Amazon EC2
- C. AWS Batch
- D. AWS Outposts

> [!success]- 答え：**A**
> Lambdaはイベント発生時にコードを実行するサーバーレスサービスで、S3イベントと組み合わせやすい。
> - A：正解。短時間のイベント駆動処理に適している。
> - B：仮想サーバーを管理する必要がある。
> - C：大規模なバッチジョブ管理に向くが、単純なS3イベント処理では過剰。
> - D：AWSインフラをオンプレミスに配置するサービス。
> **関連ノート**：[[0702_Lambda・ECS・RDS・Beanstalk比較]]

## 第4問（HTTPロードバランシング）
企業は、複数のEC2インスタンスでWebアプリケーションを実行している。HTTPヘッダーやパスに基づいてリクエストを振り分け、TLS終端もロードバランサーで行いたい。最も適切なサービスはどれか。

- A. Application Load Balancer
- B. Network Load Balancer
- C. Gateway Load Balancer
- D. AWS Transit Gateway

> [!success]- 答え：**A**
> ALBはHTTP/HTTPSのレイヤー7ロードバランサーで、パスベースルーティングやTLS終端に対応する。
> - A：正解。WebアプリケーションのHTTP/HTTPS分散に適している。
> - B：TCP/UDPなどレイヤー4の高性能分散に向く。
> - C：ファイアウォールなど仮想アプライアンスを透過的に挟む用途。
> - D：VPCやオンプレミス接続を集約するネットワークハブ。
> **関連ノート**：[[0502_Elastic Load Balancing]] / [[0503_ALB・NLB・GWLBの違い]]

## 第5問（DNS）
会社は `www.example.com` をCloudFrontディストリビューションへ向けたい。DNSレコードの管理とルーティングポリシーをAWSで行う必要がある。最も適切なサービスはどれか。

- A. Amazon Route 53
- B. AWS Certificate Manager
- C. Amazon CloudFront
- D. AWS Global Accelerator

> [!success]- 答え：**A**
> Route 53はDNS管理、ヘルスチェック、ルーティングポリシーを提供する。
> - A：正解。ドメイン名をAWSリソースへ向けるDNSサービス。
> - B：TLS証明書を管理するサービス。
> - C：CDNであり、DNSゾーン管理の主サービスではない。
> - D：固定エニーキャストIPでグローバル経路を最適化するサービス。
> **関連ノート**：[[0404_Route53]]

## 第6問（秘密情報管理）
Webアプリケーションは、データベース認証情報を安全に保存し、定期的なローテーションも行う必要がある。アプリケーションコードやAMIに認証情報を埋め込みたくない。最も適切なサービスはどれか。

- A. AWS Secrets Manager
- B. AWS Systems Manager Parameter Store
- C. AWS Key Management Service
- D. AWS Certificate Manager

> [!success]- 答え：**A**
> Secrets Managerはシークレットの保存、取得、ローテーションに適している。
> - A：正解。DB認証情報の保管とローテーションに向く。
> - B：設定値やシークレットの保存に使えるが、自動ローテーション要件ではSecrets Managerがより適切。
> - C：暗号鍵を管理するサービスで、シークレット値そのものの管理が主目的ではない。
> - D：TLS証明書を管理するサービス。
> **関連ノート**：[[1602_アイデンティティ系]]

## 第7問（EC2のAWS権限）
EC2インスタンス上のアプリケーションが、特定のS3バケットを読み書きする必要がある。長期アクセスキーをインスタンス内に保存することは禁止されている。最も適切な方法はどれか。

- A. IAMロールをEC2インスタンスプロファイルとして関連付ける
- B. IAMユーザーのアクセスキーを環境変数に保存する
- C. ルートユーザーのアクセスキーを暗号化してAMIに保存する
- D. セキュリティグループにS3アクセス許可を追加する

> [!success]- 答え：**A**
> EC2にはIAMロールを関連付け、一時認証情報でAWSサービスへアクセスさせる。
> - A：正解。長期キーを置かず、最小権限を付与できる。
> - B：長期キーの漏えいリスクがある。
> - C：ルートユーザーのアクセスキー利用は避けるべき。
> - D：セキュリティグループは通信制御であり、IAM権限は付与しない。
> **関連ノート**：[[0602_IAMロール詳細]]

## 第8問（監視）
運用チームは、EC2インスタンスのCPU使用率、アプリケーションログ、しきい値を超えた場合のアラームを一元的に扱いたい。最も適切なサービスはどれか。

- A. Amazon CloudWatch
- B. AWS CloudTrail
- C. AWS Config
- D. AWS Artifact

> [!success]- 答え：**A**
> CloudWatchはメトリクス、ログ、アラームを扱う監視サービス。
> - A：正解。性能監視とログ監視に適している。
> - B：AWS API操作の証跡を記録するサービス。
> - C：リソース設定と準拠状態を評価するサービス。
> - D：コンプライアンス文書を取得するサービス。
> **関連ノート**：[[1001_CloudWatch]]

## 第9問（監査証跡）
セキュリティチームは、AWSアカウント内で誰がどのAPIを実行したかを記録し、監査時に確認できるようにしたい。最も適切なサービスはどれか。

- A. AWS CloudTrail
- B. Amazon CloudWatch
- C. Amazon GuardDuty
- D. AWS X-Ray

> [!success]- 答え：**A**
> CloudTrailはAWS APIコールを記録する監査証跡サービス。
> - A：正解。ユーザー、ロール、サービスによるAPI操作を追跡できる。
> - B：メトリクスとログ監視が主目的。
> - C：脅威検知サービス。
> - D：分散トレーシングサービス。
> **関連ノート**：[[1002_CloudTrail]]

## 第10問（複数サービスの選択）
ある企業は、静的Webコンテンツを低遅延で世界中へ配信し、オリジンのS3バケットを直接公開しない設計にしたい。要件を満たすために使用すべきサービスまたは機能はどれか。（2つ選択）

- A. Amazon CloudFront
- B. CloudFront Origin Access Control
- C. AWS Direct Connect
- D. Amazon EFS

> [!success]- 答え：**A, B**
> CloudFrontでエッジ配信し、OACでS3への直接アクセスを制限する。
> - A：正解。世界中のエッジロケーションから低遅延に配信できる。
> - B：正解。S3オリジンをCloudFront経由に限定できる。
> - C：オンプレミスとAWSの専用線接続であり、静的コンテンツ配信ではない。
> - D：共有ファイルシステムであり、CDN配信やS3非公開化の要件を満たさない。
> **関連ノート**：[[0405_ネットワークとコンテンツ配信]] / [[0208_S3バケットポリシー]]
