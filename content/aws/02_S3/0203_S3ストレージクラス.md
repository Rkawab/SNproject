# S3 ストレージクラス まとめ

## 概要

S3のストレージクラスは、**データのアクセス頻度・耐久性・コスト**に応じて使い分けるオプション。  
全クラス共通で耐久性は **11 nines（99.999999999%）** — ただし One Zone-IA のみ例外あり。

---

## クラス一覧

| クラス                        | 用途                 | 取り出し速度 | 最低保存期間 | コスト感               |
| -------------------------- | ------------------ | ------ | ------ | ------------------ |
| Standard                   | 頻繁アクセス（デフォルト）      | 即時     | なし     | ストレージ高め・取り出し無料     |
| Standard-IA                | 低頻度・即時取り出し必要       | 即時     | 30日    | ストレージ安・取り出し課金      |
| One Zone-IA                | 低頻度・単一AZ（再作成可能データ） | 即時     | 30日    | Standard-IAより約20%安 |
| Intelligent-Tiering        | アクセスパターン不明・自動移動    | 即時     | なし     | 監視費用あり・自動最適化       |
| Glacier Instant Retrieval  | アーカイブ・四半期1回程度      | 即時     | 90日    | Standard-IAより安い    |
| Glacier Flexible Retrieval | アーカイブ・年1〜2回        | 数分〜数時間 | 90日    | 安い・取り出し速度選択可       |
| Glacier Deep Archive       | 長期保存・ほぼアクセスしない     | 最大12時間 | 180日   | S3最安値              |

---

## アクセス頻度で整理

### よく使う → Standard 系

- **Standard**：なんでもOKのデフォルト
- **Standard-IA**：月1回程度の低頻度アクセス（バックアップ等）
- **One Zone-IA**：Standard-IA のシングルAZ版。AZ消失でデータ消失リスクあり

### アーカイブ → Glacier 系

- **Glacier Instant Retrieval**：Glacier なのに即時取り出しができる
- **Glacier Flexible Retrieval**：取り出しに数分〜数時間。急がないアーカイブ向け
- **Glacier Deep Archive**：最安・最遅。コンプライアンス用途など長期保管向け

### おまかせ → Intelligent-Tiering

- アクセスパターンが読めないときに使う
- 30日アクセスなし → 自動でIA層へ移動してコスト最適化
- オブジェクト単位の監視費用がかかるため、小さいオブジェクトが大量にあると逆効果

---

## Glacier Flexible Retrieval の取り出しオプション

| オプション | 速度 | 用途 |
|---|---|---|
| Expedited | 1〜5分 | 急ぎのとき（追加料金高め） |
| Standard | 3〜5時間 | 通常 |
| Bulk | 5〜12時間 | 大量データを安く取り出す |

---

## SAA 試験頻出ポイント

- **耐久性はどのクラスも 11 nines** — ただし **One Zone-IA のみ** AZ 障害でデータ消失リスク
- **最低保存期間の課金**あり（期間未満で削除しても課金される）
  - Standard-IA / One Zone-IA：30日
  - Glacier Instant / Flexible：90日
  - Glacier Deep Archive：180日
- **Intelligent-Tiering** はオブジェクト単位の監視費用あり → 小さいオブジェクト大量保存には不向き
- **Standard** は取り出し費用なし（他クラスは取り出し課金あり）
- ライフサイクルポリシーでクラス間の自動移行が可能

---

## ライフサイクル移行の典型パターン

```
Standard
  └─ 30日後 → Standard-IA
       └─ 90日後 → Glacier Flexible Retrieval
            └─ 365日後 → Glacier Deep Archive
```

## アナロジー

| クラス | たとえ |
|---|---|
| Standard | 机の上（すぐ取り出せる） |
| Standard-IA / One Zone-IA | 引き出しの中（取り出しに少し手間） |
| Glacier Instant | 倉庫（遠いけど即座に届く） |
| Glacier Flexible | 地方の倉庫（注文してから数時間） |
| Glacier Deep Archive | 深海の金庫（取り出すのに半日かかる） |
