# AWS Data Exchange

> [!info] 外部の会社が売っている「データそのもの」を購入・定期購読できる、データ専門のお店。
> 用語が不安なら先に [[1102_データ分析の超基礎・用語集]] を。担当は分析5段階のうち**①収集**（外部購入）。

## 超かんたんに言うと

分析に使いたいデータが、自社にはないこともあります（天気・株価・人口統計など）。
Data Exchange は、そうした **外部の会社が販売しているデータセットを買って、自分の [[0201_S3]] に直接受け取れる**マーケットプレイス（お店）です。サブスク（定期購読）で最新データが自動で届く形が多いです。

> [!example] たとえ話
> Amazon.com の「**データ専門スーパー**」。
> 「気象データ定期便」を契約すると、買ったデータが自分の [[0201_S3]] という冷蔵庫に毎回自動で届くイメージ。

## どんなデータが買える？

| 分野 | 使いみち例 |
|---|---|
| 気象データ | 物流・農業の判断 |
| 金融・株価データ | 投資・市場分析 |
| 医療・ヘルスケア | 研究・治験 |
| 地理・人口統計 | 地図・マーケティング |

## 受け取り方

- 自分の [[0201_S3]] バケットへ直接配信
- API（プログラムからの取得口）経由
- データが更新されると自動で届く

## 試験での扱い

- **出題頻度は低め**。「外部のデータを購入・統合したい」というシナリオで選択肢に出る程度です。
- 受け取ったデータは [[1109_Amazon Athena]] で分析したり、[[1111_Amazon QuickSight]] で可視化したりします。
